# webfuse-core

webfuse-core是webfuse的框架核心，基础工具类。

webfuse-core是在[VJTools](https://github.com/vipshop/vjtools)的基础上进行了一些修改。


## 项目结构

``` text
.
├── constant      常量
├── exception     异常
├── kit           工具包
└── lang          扩展的核心包
```

## 使用说明

1. 在项目的pom.xml中引入：

```xml
<dependency>
    <groupId>cn.webfuse</groupId>
    <artifactId>webfuse-core</artifactId>
    <version>1.0.2-snapshots</version>
</dependency>
```
